# Tmux Cheatsheet

Prefix: Ctrl-a

## Panes
- Split vertical (left/right): prefix + v
- Split horizontal (top/bottom): prefix + s
- Move between panes: prefix + h/j/k/l
- Resize panes: prefix + H/J/K/L
- Even out panes: prefix + e
- Zoom pane: prefix + z
- Close pane: prefix + x

## Windows (Tabs)
- New window: prefix + c
- Next window: prefix + n
- Previous window: prefix + p
- Jump to window: prefix + 1..9
- Window list: prefix + w
- Rename window: prefix + ,
- Close window: prefix + &

## Sessions
- Detach: prefix + d
- Session list: prefix + s

## Copy Mode
- Enter copy mode: prefix + Escape
- Start selection: v
- Copy selection: y

## Misc
- Reload config: prefix + r
